# Welcome to Safety Radius

## Project goal 

Provide a visualization tool for the user to examin the safety of San Francisco using a safety radius and a Google Map. The user 
pick a point in the map, choose the radius and see if the region is dangerous or not.

## User profile

People that want a tool to help them to walk safely in San Francisco.

## Plataform

The proof of concept was made in using a webapp. It was not optimized for mobile neither uses reactive components.
