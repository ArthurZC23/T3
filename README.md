# Welcome to Safety Radius

## Project goal 

Provide a visualization tool for users examine the safety of San Francisco using a safety radius and a web map. Users start 
by picking a point in the map. Then, they choose the radius and a colored safety radius will be plotted, informing if the region is dangerous, a place to be careful or safe. The information is encoded in the color of the safety zone and in the audio message played when the safety radius is displayed at the map.

## User profile

People that want a tool to help them walk safe in San Francisco. The user can be color blind, as there is also sound data to inform the level of danger of the safety radius.

## Plataform

The proof of concept was made in using a webapp. It was not designed for mobile use, although it can be opened in a mobile browser.
